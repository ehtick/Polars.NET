# Polars.NET 0.3.0 Release Note
![icon](assets/icon_lite.png)
## Core

- Polars Rust Core 0.53 version adopted
- Float16(Half) supported

## API

- JoinBuildSide Enums added
- Concat for DataFrame and LazyFrame added strict and unitLengthAsScalar parameters
- All explode methods now have emptyAsNull and keepNulls parameters

## IO

- DeltaLake CRUD and maintaince features added
- Cloud(Aws,Azure,GCP) supported

## Runtime

- Native Libraries now are separated from core library.
